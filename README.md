* https://shierro.github.io/cv
* Built with Vuejs + Vuex with Vuetify

# Install

`
$ git clone https://github.com/shierro/cv
$ npm install
`

# Run unit test
`
$ npm test
`

# Run e2e test
`
$ npm run e2e
`

# Build prod files in /docs folder for github page
`
$ npm run build
`
