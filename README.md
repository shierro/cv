# Install

`
$ git clone https://github.com/shierro/cv
`

`
$ cd cv && npm install
`

# Run unit test
`
$ npm test
`

# Run e2e test
`
$ npm run e2e
`

# Build prod files in /docs folder
`
$ npm run build
`
